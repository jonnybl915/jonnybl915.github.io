The Yada

The Yada is a tool with which users can view and generate synopses of articles across the web. My team and I developed a Google Chrome Extension for viewing and creating these "Yadas" and a website for displaying and sorting them. I worked alongside one fellow backender and one frontender to make it happen. Tech used on the backend: Java, Spring MVC with Hibernate, Jsoup, and Postgresql

APitunes

A platform to help local musicians exhibit their music. Artists can create profiles and upload music for other users. Non-artist users log in to listen and vote on the music. Built the backend of the web app with Spring MVC and Hibernate, and an H2 database structure.

Skip To The Loo

This app allows users to rate and subit the locations of public restrooms in their area. The user submits their rating along with the latitude and longitude of the restroom location for other users' benefit. Tech used: Java with Sparc microframework, JUnit for testing, and H2 for database structure
